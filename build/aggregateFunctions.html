Aggregate Functions

General command:

SELECT aggregate(columnCondition) AS `alias`
FROM table alias ;

Aggregate:

The variable aggregate can have one out of five values. It could be count, min, max, avg or sum. Count gives out the number of values in the column columnCondition, min gives out the smallest value of the column, max the biggest, avg the average and sum the sum of all values in the column.

Alias:

With the Keyword AS you can define analias to the min, max, avg, count or sum value of the column. It can be used in the same way an alias of an database table is used.

Example:

In this example we have a database table called accounts with the columns id, name, surname, birthday, email, phoneNumber and password. We want to display the number of accounts in the database. In order to do this we write the following command:

SELECT count(*) AS `Number of accounts`
FROM accounts a;
