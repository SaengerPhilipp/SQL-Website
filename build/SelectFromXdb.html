Multi DB Select

General command:

SELECT columnConditionDB1, columnConditionDB2, columnConditionDB3
FROM table1 alias1, table2 alias2, table3 alias3
WHERE conditionConnect1
AND conditionConnect2
AND condition
ORDER BY orderColumn direction ;

Explanation:

In order to display data out of multiple database tables that are all connected to one of them you apply the same principles as to display two connected database tables. Instead of two columnConditions you have x ones and instead of two table names and aliases you have x ones. You also need one columnConnect for each table you wat to conect to the base table.

Example:

In this example we have a database table called accounts with the columns id, name, surname, birthday, email, phoneNumber and password. We want to display every account where te name starts with an A and te person is born befor the year 2000. We also want to order the table alphabetically descending by the column name. We also have a secnd database table called orders which is connected to the previous one. It has the columns id_acc (which is the same as id in th accounts table) and orderedproducts. Now we also want to display which products the accounts ordered. An addition to that we have athird database table called subscribers which has the columns id_acc (which is the same as id in th accounts table) and subscriptionmodel. The dta out of this database should also be displayed. In order to do this we write the following command:

SELECT a.name, a.surname, a.birthday, o.orderedproducts, s.subscriptionmodel
FROM accounts a, orders o, subscribers s
WHERE 'a.id' = 'o.id_acc'
AND 'a.id' = 's.id_acc'
AND a.name LIKE 'A%'
AND a.birthday < '2000-01-01'
ORDER BY a.name DESC;
