Two DB Select

General command:

SELECT columnConditionDB1, columnConditionDB2
FROM table1 alias1, table2 alias2
WHERE conditionConnect
AND condition
ORDER BY orderColumn direction ;

Select From:

In order to display data out of two database tables that are connected you need to give the information of both tables into the SELECT FROM part of the command. that means you add a columnCondition for the second database table and you also add the name of this table with an alias behind the FROM.

conditionConnect:

If two database tables are connected there is one column in each of those tables that is the same. You have to create a condition that says that the column of the one database has to equal the one of the other. In general it would look like this: 'alias1.columnDB1' = 'alias2.columnDB2'

Example:

In this example we have a database table called accounts with the columns id, name, surname, birthday, email, phoneNumber and password. We want to display every account where te name starts with an A and te person is born befor the year 2000. We also want to order the table alphabetically descending by the column name. We also have a secnd database table called orders which is connected to the previous one. It has the columns id_acc (which is the same as id in th accounts table) and orderedproducts. Nowwe also want to display which products the accounts ordered. In order to do this we write the following command:

SELECT a.name, a.surname, a.birthday, o.orderedproducts
FROM accounts a, orders o
WHERE 'a.id' = 'o.id_acc'
AND a.name LIKE 'A%'
AND a.birthday < '2000-01-01'
ORDER BY a.name DESC;
