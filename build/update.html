Update

General command:

Update `DB`.`table`
SET
`valueColumn` = `value`
WHERE
(condition);

Update:

The variable valueColumn equals the name of the column one wants to update data in and the variable value equals the new value of that column. The variable condition equals condition to define the specific row one wants to change data in.

Example:

In this example we have a database called onlineshop with a table called account with the columns name, surname, and email. We want to change the email of Max Musterman to the email max.musterman@gmail.com. In order to do this we write the following command:

UPDATE `onlineshop`.`account`
SET
`email` = `max.mustermann@gmail.com`
WHERE
(`name` = `Max`);
