Delete

General command:

DELETE FROM `DB`.`table`
WHERE
(condition);

Example:

In this example we have a database called onlineshop with a table called account with the columns name, surname, and email. We want to delete the account of Max Musterman. In order to do this we write the following command:

DELETE FROM `onlineshop`.`account`
WHERE
(`name` = `Max`)
AND
(`surname` = `Musterman`);
